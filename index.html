Tentang SIGETA

Sistem Informasi dan Monitoring Greenhouse Terintegrasi AI

Mengenal SIGETA

SIGETA adalah platform cerdas yang menggabungkan teknologi Internet of Things (IoT) dengan kecerdasan buatan untuk memantau dan mengoptimalkan kondisi lingkungan di dalam greenhouse.

Sistem ini menggunakan berbagai sensor untuk mengumpulkan data real-time tentang suhu, kelembapan udara, kelembapan tanah, intensitas cahaya, dan kualitas udara.

Data tersebut kemudian dianalisis oleh algoritma AI untuk memberikan rekomendasi otomatis guna menciptakan lingkungan tumbuh yang optimal bagi tanaman.

Dengan SIGETA, petani dan peneliti dapat memantau kondisi greenhouse dari mana saja melalui antarmuka web yang responsif, serta menerima notifikasi jika terdapat kondisi yang tidak normal yang memerlukan perhatian.

Informasi API

Sistem ini terhubung dengan API backend yang menyediakan data sensor real-time. Berikut adalah contoh struktur data yang diterima:

{
  "time": "2023-11-15 14:30:45",
  "temperature": 25.6,
  "humidity": 68.2,
  "mq": 145,
  "status": "✅ AMAN"
}

Endpoint API:

  • POST /data - Menerima data sensor dari ESP32
  • GET /latest - Mengambil data terbaru untuk ditampilkan di web
